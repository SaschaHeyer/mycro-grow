🍄 Mushroom fruiting conditions — 14-species reference

Colonization and fruiting temperature, humidity, CO₂, fresh-air exchange, time to fruit, and typical biological efficiency for 14 gourmet & functional mushroom species. This is the FRUITING_CONDITIONS dataset from the MIT-licensed mycro-grow library — the same data behind the interactive Fruiting Conditions cheat-sheet at usemycro.com.

These are typical cultivation ranges compiled from common practice — useful starting points for dialing in a fruiting room, not guarantees. Conditions interact: pushing CO₂, humidity and fresh-air together is what actually shapes a flush. For the why behind the yield column, see biological efficiency, explained; to plan a specific batch, use the Grow Calculator.

Use the data

The full dataset — including each species' grouping, difficulty, and a practical grower note — is available as open JSON and from the library:

# raw open data
curl https://saschaheyer.github.io/mycro-grow/species.json

# or from the library
import { FRUITING_CONDITIONS, SPECIES_BE } from 'mycro-grow';
FRUITING_CONDITIONS.length;        // 14
FRUITING_CONDITIONS[0].fruitC;     // [10, 21]  (Blue Oyster, °C)
